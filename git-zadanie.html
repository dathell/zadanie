ZADANIE GIT

W internecie jest dużo materiałów o tym czym jest git i jak z niego korzystać. Jest też pare serwisów pozwalających na udostępnianie swojego kodu. Jednym z nich jest Github(https://github.com/) i chciałbym aby twoim pierwszym zadaniem było założenie konta a następnie udostępnienie twojego kodu ze stroną właśnie na tym serwisie.

Jakbyś miał z tym problemy to daj znać.

Do zrobienia:

1. Załóż konto na github

2. Po zalogowaniu stwórz nowe zdalne repozytorium gitowe

3. W pliku gdzie masz projek zainicjalizuj repozytorium lokalne: git init

4. Dodaj wszystkie pliki do śledzenia: git add .

5. Zrób commita: git commit -m "First commit"

6. Połącz repozytorium lokalne ze zdalnym: git remote add origin

7. Wypchnij aktualną wersję: git push -u origin master
